REST-сервис, который возвращает id предметов заданного цвета(color) содержащиеся в ящике c заданным идентификатором (box) с учётом того, что в ящике может быть ещё ящик с предметами требуемого цвета.

Для этого нужно отправить POST-запрос с телом запроса в JSON вида:

  • POST /test HTTP/1.1
  • Host: localhost
  • Accept: application/json
  • Content-Type:application/json
  • Content-Length: 25
  • {"box":"1","color":"red"}
